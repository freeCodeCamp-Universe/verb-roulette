# Verb Roulette - README

**Verb Roulette** is a browser-based drill for practicing irregular verb forms — spin the wheel to get a verb, then type all three forms from memory.

## Key Features

**Core Gameplay**: Players spin a roulette wheel that lands on a random irregular verb. They then type the infinitive, past simple, and past participle before the timer runs out.

**11 Languages**: Verbs are available in two language groups — Germanic (English, German, Dutch, Swedish, Danish, Norwegian, Icelandic) and Romance (French, Spanish, Portuguese, Italian) — switchable from the start screen.

**Timer Modes**: Players can set a per-verb time limit of 10 s, 30 s, 60 s, or unlimited, adjusting the challenge to their level.

**Score and Streak Tracking**: Each session tracks score, current streak, and round count, rewarding consistent correct answers.

**Grammar Reference**: A built-in modal explains the verb form rules for the selected language, including tense formation and irregular patterns.

**Verb List**: A browsable table of all irregular verbs for the active language, accessible without leaving the game.

## Tech Stack

Vanilla HTML, CSS, and JavaScript. No build step or dependencies — open `index.html` directly in a browser.
